# Trade Titans
A CSC207 project @UofT

### So, what's this?
Trade Titans is a stock exchange software that features: 
1. Exchange of stocks, currencies and cryptos
2. Multiple portfolios in different currencies
3. Track holding with real-time and historical market data
4. Trade at real-time market price
5. View historical transactions (of different portfolios)
6. View the interactive graphical performance history of stocks and currencies
